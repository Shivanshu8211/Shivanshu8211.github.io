## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/Shivanshu8211/My-Portfolio.git
cd my-portfolio

# Install dependencies
npm install

# Start local development server
npm run dev

```

## 🚀 Deployment (GitHub Pages)

```bash
# In vite.config.js
# Add " base: '/my-portfolio/' " in "defineConfig"

# Build the project
npm run build

# Deploy
npm run deploy

```
